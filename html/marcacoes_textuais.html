Marcações textuais

Dentro dos textos possuimos diversas formas de classificar uma palavra, por exemplo; às vezes ela se refere à um endereço ou um nome de uma marca.

Além de classificar, às vezes queremos dar mais importância a uma palavra, por exemplo; deixando-a em negrito.

Tag b e strong

Torna o texto em negrito, o "strong" além de deixar em negrito serve para indicar aos sistemas de busca que o conteúdo tem uma relevância maior

b
strong

Tag i e em

Torna o texto em itálico, o "em" além de deixar em itálico serve para indicar aos sistemas de busca que o conteúdo tem uma relevância maior

i
em

Tag mark

Deixa o texto grifado

Tag small

Deixa o texto pequeno

Tag del e ins

Deixa o texto com uma linha, uma linha no meio do texto significa que ele foi deletado, para esse caso usamos "del", uma linha em baixo do texto significaque ele foi inserido, para este caso usamos o "ins"

Texto delete deletado e inserido, usa-se o del para marcar textos que foram removidos, e ins para textos que foram inseridos, não os utilize simplesmente para colocar um traço nas palavras pois ambos possuem uma função para leitores de tela

Atributos que podem ser inseridos na tag:

  • cite (cita uma url a qual deverá conter o motivo para tal texto ser removido ou inserido, por exemplo um site de dicionário)
    • Url
  • datetime (marca a data de modificação @@@@@ verificar GMT)
    • Valor YYYY-MM-DDThh:mm:ss:
      YYYY-MM-DDThh:mm:ss
      YYYY - define o ano
      MM - define o mês
      DD - define o dia
      T - está sendo usado para separar informação,
      neste caso separa a data da hora
      hh - define as horas
      mm - define os minutos
      ss - define os segundos

Tag sub e sup

Deixa o texto subscrito (sub) ou superescrito (sup), em ambos os casos o texto se torna menor, em uma ocasião ele fica mais para baixo e em outra mais para cima

Tags q e blockquote

Insere aspas em uma palavra, o q adiciona aspas, já o blockquote além de adicionar aspas ele destaca o texto.

Exemplo da tag q: Texto para demonstração

Exemplo da tag blockquote:

Atributos que podem ser inseridos na tag:

  • cite - serve para citar a url de onde foi tirado o determinado trecho.
    • Url

Tag abbr

Marca uma abreviação, por exemplo HTML pode ser marcado, pois ele é uma abreviação de Hyper Text Markup Language

Atributos que podem ser inseridos na tag:

  • title - título como elemento, geralmente são mostradas como um texto de dica de ferramenta quando o mouse se move sobre o elemento, revelando assim uma pequena caixa de texto.

Tag cite

Marca algo como citado, algo famoso que possui um nome próprio, como por exemplo; a obra Monalisa ou a música Highway to Hell da banda AC/DC

Tag code

Marca um texto sendo escrito em código de computador, exemplo:

var x = "Hello World"
window.alert(x)

Tag kbd

Marca uma keyboard, tecla de entrada, indica uma tecla que o usuário pode pressionar, por exemplo; clique a tecla "t"

Tag var

Marca uma variável de uma equação

x + y = 2
y + y = 3

Tag samp

Indica uma mensagem de resposta de um programa, clique no botão abaixo para fazer o texto aparecer:

Testando... clique no botão para esse texto sair

Tag bdo e bdi

Modifica a direção textual

Atributos que podem ser inseridos na tag bdo:

  • dir
    • ltr
    • rtl

Tag address

Define as informações de contato (autor / proprietário) de um documento ou artigo.
