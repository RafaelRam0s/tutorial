Imagens

Podemos colocar imagens na página de uma forma bem simples, devido aos vários exemplos que terão que ser dados os guardamos dentro de vários details, se quiser ver como funcionam adicione o atributo open no details

Tag img

Para colocarmos uma imagem no site precisamos da tag img e definir o atributo src

Atributos como src, alt, width e height são atributos essenciais para se colocar na tag.
O src é quem vai carregar o arquivo, referenciando o local do arquivo sendo armazenado localmente ou em site externo, se tiver uma imagem em base64 (código de diversos dígitos que transformam uma imagem em texto) você pode colocar o código também.
O atributo alt é um texto que só aparecerá caso ocorra algum problema no carregamento do arquivo da imagem, mas esse atributo além de aparecer como texto quando dá problema no carregamento da imagem ele também é um texto alternativo que é lido para deficientes visuais, então é recomendável que no alt esteja escrito o que a imagem representa.
Os atributos width e height são importantes de serem definidos, pois antes da imagem ser carregada o elemento só ocupa uma linha, e quando a imagem é processada pelo site ela altera a posição dos elementos seguintes, mas se definir uma altura e largura fixa o espaço já fica reservado.

Tome cuidado com o peso de arquivo da imagem, por exemplo pense que temos uma imagem que pesa 5 megabytes, está na extensão png e possui dimensões 2400X1200, quando a sua página for carregada a imagem será baixada, 5 megas é muita coisa para uma imagem em sites na web e pode causar uma demora no carregamento da página, algumas maneiras de diminuir o peso da imagem é; utilizando compressores de arquivo, diminuir as dimensões da imagem e mudar a extensão, para por exemplo webp, lembrando que esses métodos podem fazer a qualidade da imagem decair. Além do peso do arquivo é preciso ficar de olho nas dimensões da imagem, se você for utilizar uma imagem de 2400 por 1200 em um bloco de 200 por 100, altere as dimensões do arquivo para que seja processado adequadamente

Tag picture e source

Muitas vezes é preciso para que em tamanhos de telas diferentes coloquemos imagens mais adequadas para as dimensões da tela do aparelho, por exemplo, um celular tem uma tela de 400px e o notebook uma tela de 1200, uma imagem de 1000 de largura serve perfeitamente para o computador, mas para o celular fica muito pesada, sendo que poderia ser otimizada, e para trazermos a imagem correta de acordo com o tamanho de tela utilizamos os elementos picture e source

Com o picture criamos um elemento que armazenará vários elementos source e 1 elemento img, no img colocamos a imagem que será para a menor tela ou para a maior tela

Nos elementos source informamos 2 atributos essenciais, o media e o srcset. O srcset é o link, caminho ou código em base64 alternativo, que substituirá o src do elemento img. O media serve para falar a partir de que tamanho de tela a imagem que está no srcset deverá ser carregada e substituir o elemento img.

Quando utilizamos este tipo de forma responsiva para alterarmos o tamanho da imagem, devemos sempre pensar se queremos usar como base a tela do celular com uns 400px ou a tela de um computador com uns 1600px, a escolha de um deles mudará a forma com a qual ajustamos os elementos source. Se for com base no computador, escrevemos primeiro o menor source até o maior, colocando dentro do atributo media o valor "(max-width: px;)" e antes de px inserimos o tamanho da tela. Se for com base no celular, escrevemos primeiro o maior source até o menor, colocando dentro do atributo media o valor "(min-width: px;)" e antes de px inserimos o tamanho da tela.

Tag map e area

Algumas vezes queremos que uma parte da imagem seja clicável, para que possamos interagir com a imagem, seja redirecionando ou utilizando conjuntamente com um evento javascript, diferente de utilizar o elemento "a" para criar um hyperlink em toda a imagem é possível que somente um elemento visual da imagem seja acessado.

Como fazer:
1° Crie a imagem que você usará como base, faça ela padrão com o src, alt, width, height e os outros atributos que você precisa configurar.
2° Crie um elemento "map"
3° Adicione no elemento "map" o atributo "name", o valor deve ser um nome único na página
4° Dentro do elemento map adicione elementos area
5° No elemento area é preciso adicionar atributos como, shape, coords
O shape definirá como devem ser escritas as coordenadas, por padrão recomendo utilizar apenas 2 valores, o circle (círculo) e o poly (polígono), o círculo você precisa indicar o ponto do centro do círculo e depois o raio, para o polígono você vai precisar indicar uma coordenada para cada vez que o traço da figura mudar de direção.

Apesar de com esses elementos ser possível fazer uma área da imagem ser clicada, ele não é responsivo, logo, só funciona com imagens que na página estão com seus width e height fixos
