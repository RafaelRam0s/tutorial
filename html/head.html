Estrutura HTML

No HTML começamos informando qual a versão que o "leitor de código html" deve utilizar, estamos aprendendo HTML5, logo devemos informar: !DOCTYPE html para que o arquivo seja lido na versão 5.

Tag html

Abrimos a tag html para que possamos escrever códigos html.

Atributos que podem ser inseridos na tag:

  • lang (idioma - informa o idioma da página para que sistemas de tradução possam automaticamente saber de qual idioma é baseado, para que quando um usuário pedir a tradução da página o mecanismo traduza com maior facilidade)

    • Abreviações de idiomas (se a página é em portugues do Brasil colocamos o valor "pt-br", se for em inglês colocamos "en" e assim por diante)

  • dir (direção textual - informa qual a direção que os caracteres devem ser lidos)

    • ltr (esquerda para direita: formato de leitura ocidental)

    • rtl (direita para esquerda: formato de leitura oriental)

Tag head

Dentro dessa tag inserimos informações que serão aplicadas em toda a página e informações para sistemas de busca, como por exemplo; o Google.

Tag meta

Informa aos sistemas como a página deve ser lida

Atributos que podem ser inseridos na tag:

  • charset (escolha de caracteres - informa quais caracteres serão lidos, se não informado caracteres especiais, como cecedilha "ç", são substituídos por códigos não agradáveis)

    • UTF-8

  • http-equiv

    • refresh

    • X-UA-Compatible

  • content (conteúdo - a partir de atributos como "name" e "http-equiv" pode ter valores que realizem determinadas ações)

    • texto

        Tagname http-equiv

      • Com valor refresh, faz com que o content possa definir em quantos segundos a página, deve atualizar ou enviar para outro site, exemplo; content="100 ../index.html"

      • Com valor X-UA-Compatible, informa a compatibilidade com outros sistemas, exemplo; content="IE=edge"

        • Tagname name

      • Com valor viewport, definimos qual o formato que a tela deve ser visualizada, normalmente queremos que em dispositivos menores a tela continue na mesma proporção de tela que veríamos em um computador, então podemos utilizar o seguinte valor; content="width=device-width, initial-scale=1.0"

      • Com valor author, no content podemos escrever o nome do autor da página html

      • Com valor description, informamos uma descrição do site que pode ser utilizada como meio de resumo das informações que serão apresentadas no site

  • name (nome - define informações para sistemas de busca e por leitores de telas)

    • viewport (Visualização da página - define no content qual o formato que a tela deve ser visualizada, normalmente queremos que em dispositivos menores continue com a proporção da tela de um pc então podemos escrever: content="width=device-width, initial-scale=1.0" se isso não for escrito alguns comportamentos de layout inesperados podem surgir)

    • author (autor - define quem escreveu a página)

    • description (descrição - quando pesquisamos por um site no Google podemos ter uma leitura prévia do conteúdo do site, assim que definimos este atributo mudamos o que pode ter na descrição)

    • keywords (palavra-chave - ao escrever palavras no content, separando cada palavra com vírgula, você ajuda a indexar melhor sua página em sistemas de busca)

Tag link

Trazem arquivos de estilo, como por exemplo folhas de estilo e ícones

Atributos que podem ser inseridos na tag:

  • rel (relacionamento, avisa ao navegador como deve ser tratado o documento externo que será vinculado)

    • icon (icone - adiciona no título uma imagem)

    • stylesheet (folha de estilo - traz um arquivo css para a página)

  • type (tipo, informa o formato em que o arquivo que está sendo trazido está)

    • text/css

    • image/png

    • image/jpeg

  • href

    • url

Tag base

Define um caminho de arquivo padrão por onde seus links irão começar. Por exemplo, as rotas de arquivo são definidas a partir de onde o arquivo está armazenado, mas é possível definir a rota padrão para começar na raiz ou em outra pasta, por exemplo; base href="https://www.algum_dominio_que_possua.dominio/html/" target="_self"

Demonstrações de funcionalidade:

Exemplo:

  • base href="https://www.site_do_ramos.com/html/"
  • img src="_icone_r.ico"
  • Então no src do img é como se estivesse escrito; https://www.site_do_ramos.com/html/_icone_r.ico

Exemplo 2:

  • base href="https://www.site_do_ramos.com/html/"
  • link rel="stylesheet" href="../css/fundo.css"
  • Ficaria https://www.site_do_ramos.com/css/fundo.css

Atributos que podem ser inseridos na tag:

  • href (configura onde as urls irão usar o arquivo base para encontrar um item)

    • url

  • target (configura como abrir os links)

    • _blank

    • _parent

    • _self

    • _top

    • framename
