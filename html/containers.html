Tag meter

Uma barra de medição escalar dentro de uma faixa conhecida ou um valor fracionário. Isso também é conhecido como medidor.

2 out of 10

Atributos que podem ser inseridos na tag meter:

  • min - mínimo, junto do atributo max configura um alcance para o quanto deve ser preenchido ao informar um valor no atributo value
  • max - máximo, junto do atributo min configura um alcance para o quanto deve ser preenchido ao informar um valor no atributo value
  • low - baixo, caso o value esteja abaixo do valor informado neste atributo a cor mudará
  • high - alto, caso o value esteja acima do valor informado neste atributo a cor mudará
  • optimum - aumenta/diminui os graus de cores a partir do valor informado
  • value - valor
