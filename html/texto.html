Texto em HTML

Para escrevermos algo, seja em um livro, uma redação ou um código, possuimos regras e métodos de organização, a forma de montar um texto muitas vezes é composta por um título e um parágrafo, e neles definimos como e onde haverá quebra de linha e como os espaçamentos entre as palavras devem ser considerados.

Tags h1, h2, h3, h4, h5 e h6

Você pode fazer um título na página com as tags h1, h2, h3, h4, h5 e h6, todas representam um título, a diferença em usar cada uma delas é a importância que o título tem.

Por exemplo um livro teria o título da capa em h1 pois ele é único e muito importante para explicar o conteúdo, dentro dele temos o sumário, história e prefácio, esses seriam o h2 e dentro da história temos os capítulos com seus títulos esses seriam o h3.

O ideal é que por cada página html exista uma única tag h1, ela é importante para os mecanismos de busca como o Google poderem entender e classificar melhor o seu site.

Tag p

A tag p representa um parágrafo, é dentro dele que escrevemos a maior parte do conteúdo textual das páginas

Tag br

No código html podemos dar vários espaços, enters, tabs mas se as tags estiverem com sua abertura e fechamento corretas o texto sempre ficará formatado para ser uma linha inteira com quebras de linha automaticas e espaços únicos entre as palavras.

A tag br representa uma breaking row, que no português significa quebra de linha, ela realiza o ato de onde estiver inserida no texto causar uma quebra de linha.

Tag wbr

Quando escrevemos muitas palavras em um texto é normal que quando se chega ao final da página a palavra seja dividida, mas no caso do html cada palavra age como um bloco, então para que ocorra uma divisão em uma palavra muito extensa ao encontrar o final da linha temos que adicionar a tag "wbr" e caso a página passe pela palavra ela será dividida.

Tag pre

No código html podemos dar vários espaços, enters, tabs mas se as tags estiverem com sua abertura e fechamento corretas o texto sempre ficará formatado para ser uma linha inteira com quebras de linha automaticas e espaços únicos entre as palavras.

Porém com a tag pre o html passa a interpretar o que escrevemos como literal, então quebras de linha, espaços, tabs e enters são tudo manual, não existe nem quebra de linha automatica.
