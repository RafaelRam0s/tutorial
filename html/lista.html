Listas

Quando queremos enunciar uma série de itens utilizamos uma lista

Tags ul, ol, li

Para criarmos uma lista primeiro precisamos informar se ela será uma lista ordenada ou desordenada, uma lista ordenada é aquela que queremos numerar a quantidade de itens que estão presentes, já uma lista desordenada é aquela que só precisamos colocar os itens

Exemplo de lista desordenada - Lista de mercado

  • Pão
  • Manteiga
  • Suco de laranja

Exemplo de lista ordenada - Tarefas realizadas no dia

  1. Acordou
  2. Comeu
  3. Dormiu

Atributos que podem ser inseridos na tag li quando em uma lista ordenada:

  • value - define o número do item, por exemplo temos os itens: 1, 2 e 3, no item 2 podemos no elemento li definir o valor para 5, isso fará com que a partir daquele ponto os próximos itens sejam enumerados a partir do cinco, logo nessa lista exemplo ficaria com os itens: 1, 5 e 6
    • Número

Item de lista com value:

  1. Lorem
  2. Lorem
  3. Lorem

Atributos que podem ser inseridos na tag ol:

  • start - define a partir de que número deve começar a contagem dos itens de lista
    • Número
  • reversed - Faz com que a contagem seja feita do último item até o primeiro
