@@@@@ Menu Completo

Deverá ter uma lista de escolhas, uma ao lado da outra

Algumas dessas listas de escolhas deverão conter outra lista de opções, que deverá aparecer ao ser posicionado o mouse em cima

Uma dessas listas de escolhas, deverá ser um conteiner das listas de escolhas que acabem por criar uma segunda linha de elementos no menu, deve ficar posicionado no canto direito, este elemento deverá ser clicável, quando clicado deve ocupar toda a tela e apresentar todos as escolhas do menu

Caso a página diminua ou aumente o menu deve continuar com uma linha e as escolhas que acabem por criar outra linha no menu devem ser alocadas dentro do conteiner de listas na seguinte ordem, o elemento mais a direita na linha de menu deverá ficar por último na lista do conteiner

A escolha que tiver uma lista de opções, caso passe para dentro do conteiner, deverá ser clicável para visualizar as opções, e se clicar de novo as feche
