Exemplos

rgb - o valor rgb aceita 3 parametros cada um varia de 0 a 255 esse valor refere-se a quantidade de presença da cor vermelha, verde e azul, respectivo a cada parâmetro, exemplos: "rgb(255, 255, 255)", "rgb(255, 0, 0)", "rgb(0, 255, 0)", "rgb(0, 0, 255)", "rgb(0, 0, 0)", "rgb(120, 40, 200)"

rgba - igual o valor rgb só que aceita um quarto parâmetro, o alpha, que diz a quantidade de translucidez do elemento, o parâmetro varia de 0 a 1, então 0.5 seria metade da translucidez, exemplos: "rgba(255, 255, 255, 0)", "rgba(255, 255, 255, 1)", "rgba(255, 255, 255, 0.5)", "rgba(255, 255, 255, 0.8)"
